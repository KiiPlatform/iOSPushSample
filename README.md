# iOSPushSample

## Requirements:
* iPhone/iPad (iOS 5.0 or later)
* Xcode (version 4.2 or later)

## How to Install
1. Setup application environment
* Create Push-SSL certificate (p12)
* Create a application in Kii DeveloperPortal
* Upload p12 file and set password

2. Build and install application
* Open iOSPushSample.xcodeproj
* Open /Supporting Files/iOSPushSample-Prefix.pch
  * Edit APPID/APPKEY
* Launch on iOS devices
